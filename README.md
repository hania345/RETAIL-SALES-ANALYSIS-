Project Overview

Developed an interactive Retail Sales Dashboard in Power BI to analyze sales performance, 
customer purchasing trends, product revenue, payment methods, shipping costs, and product returns. 
The dashboard enables users to monitor key business metrics through interactive visualizations and filters.

Tools Used

Power BI
Microsoft Excel
SQL Server (for data preparation)
Power Query

Dashboard Features

KPI Cards (Sales, Orders, Quantity, Returns, Shipping Cost, Average Discount)
Sales Trend Analysis
Revenue by Product
Payment Method Distribution
Quantity Sold by Product
Regional Return Analysis
Product Performance Table

Key Insights

Total Sales: 4.38M
Total Orders: 1,500
Total Quantity Sold: 15.6K
Average Discount: 7%
Total Returns: 372
Shipping Cost: 41.26K
Tablets generated the highest revenue.
The East region recorded the highest number of returns.
Online payments were the most frequently used payment method.
